📊 Company Sales Analysis using NumPy

A beginner-friendly NumPy mini project that demonstrates how numerical computing is used in real-world data analysis.
This repository focuses on analyzing a company’s monthly sales data using NumPy’s core features.

🚀 Project Overview

This project simulates a real business scenario where sales data is analyzed to extract meaningful insights.

You will:

Work with numerical data using NumPy arrays

Perform statistical analysis without loops

Identify business trends such as best and worst sales months

This is designed for data science beginners who want hands-on practice with NumPy.

🧠 Why NumPy?

Python lists are fine for small tasks. For data analysis, they’re weak.

NumPy is used because it is:

Faster and memory-efficient

Vectorized (operations without loops)

Packed with built-in mathematical & statistical functions

Capable of handling multi-dimensional data

NumPy also forms the foundation for advanced libraries like Pandas and machine learning frameworks.

📌 What This Project Does

The program analyzes 12 months of sales data and computes:

✅ Total annual sales

✅ Average monthly sales

✅ Highest and lowest sales values

✅ Best and worst performing months

✅ Months performing above and below average

All calculations are performed using NumPy arrays, not Python loops.
